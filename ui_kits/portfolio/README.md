# Portfolio UI Kit — Sanjit Shrestha

A high-fidelity React recreation of the rebranded portfolio. Components are intentionally simple (cosmetic, not production); they exist to demonstrate the system in motion and serve as copy-paste building blocks.

## Files

- `index.html` — the assembled portfolio (single-page) using all components below
- `Nav.jsx` — top navigation with mono links and status pill
- `Hero.jsx` — editorial hero: serif display, mono caption, key/value bio
- `StatBlock.jsx` — 4-up tabular stat row
- `WorkList.jsx` + `WorkRow.jsx` — index-numbered project list (the rebrand's signature element)
- `StackGrid.jsx` — categorized tech tags
- `ExperienceTimeline.jsx` — vertical timeline with hairline rule
- `ContactCard.jsx` — key/value metadata + CTA cluster
- `Footer.jsx` — minimal mono footer
- `ui.jsx` — primitives (Button, Tag, Eyebrow, KV, StatusDot)

## Notes

This is a **single-product** UI kit — Sanjit's work is one portfolio site. We've covered the screens that matter: the home/index page (hero, work, stack, experience, contact). A "case study" detail page is out of scope for this pass; flag if needed.
